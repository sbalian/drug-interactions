Drug Interactions: Appearance in Same Report

By searching the openFDA Adverse Drug Events database, this app finds the number of reports for which a given drug appears with a set of other drugs. Only looks for fatal cases submitted by physicians.

Type in a drug's generic name and generic names of other drugs.
